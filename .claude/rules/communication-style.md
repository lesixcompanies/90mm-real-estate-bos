# Communication Style

*This file is populated during the initial diagnostic based on how the agent
actually communicates. The deployment prompt observes the agent's tone,
vocabulary, formality level, and patterns during the conversation and fills
this file in Phase 3.*

## Instructions for the Deployment Prompt
When populating this file, observe and document:
- Tone: formal, casual, direct, roundabout, analytical, emotional?
- Register: does the agent use technical language, plain language, or a mix?
- Vocabulary: any industry-specific terms they use naturally?
- Pace: do they give long, structured answers or short, clipped ones?
- What to avoid: any patterns that would feel off or patronizing?
- What works: what kind of communication would this agent respond to best?

Keep this file short — it should read like a quick reference, not an essay.
