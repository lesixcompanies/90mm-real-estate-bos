# Negative Branch Reservation (NBR)

## Purpose
Every change creates the possibility of new problems. The NBR asks:
"If we do this, what could go wrong?" — and then finds ways to prevent
those negative outcomes without abandoning the direction.

## Process
1. Review the FRT for any point where the chain could produce an
   unwanted outcome
2. Ask the agent directly: "What concerns do you have about this
   direction? What could go wrong?"
3. For each negative branch identified:
   a. Name the negative outcome specifically
   b. Trace the cause-and-effect: why would the proposed change
      lead to this outcome?
   c. Identify the trimming action: what additional step would
      prevent the negative outcome while preserving the positive
      direction?
   d. Validate that the trimming action doesn't create its own
      negative branch (if it does, trim that too)
4. Present each trim to the agent and confirm it addresses their concern

## Common Negative Branches in Real Estate Agent Practices
- "If I block time for content, I'll miss a client call" →
  Trim: Set up a professional voicemail/auto-response that
  promises callback within 2 hours. Test whether any client objects.
- "If I send postcards, people will think I'm soliciting" →
  Trim: Frame as neighborhood resource, not sales pitch. Lead
  with value (spring cleaning tips), not services.
- "If I define the partner handoff, it might create tension" →
  Trim: Frame as protecting both parties. Present as "what happens
  when we get our first deal" rather than "what you're doing wrong."
- "If I invest in this and get no listings, I've wasted money" →
  Trim: Calculate actual cost. Three postcards to 150 homes is
  probably under $500/year. Compare to what agents spend on lead
  gen that doesn't work.

## How to Present It
After the FRT, say: "That's the direction and what it should produce.
Now let's pressure-test it. What could go wrong? What concerns do
you have?"

Address each concern as a legitimate reservation, not a fear to
dismiss. Find the trim. Validate the trim doesn't create new problems.

## Resistance Layer Addressed
- **Layer 5: "Yes, but the solution has negative ramifications."**
  Every concern raised is Layer 5 until resolved.
  Process all concerns. Trim all branches.
  If a branch cannot be trimmed → Return to the EC and challenge a
  different assumption. The direction may need to change. This is
  not failure; it's rigor.
  ALL negative branches must be resolved before advancing.

## Output
- Each negative branch identified
- The trimming action for each
- Agent's assessment of whether the trims are sufficient
- Save to diagnostics/ as part of the change sequence record

## What Happens Next
Once all significant negative branches are trimmed and Layer 5 is
resolved, the direction is validated. Move to Strategy & Tactics
Tree to build the execution sequence.

## Critical Rule
If a negative branch cannot be trimmed — if the proposed direction
genuinely creates a problem that can't be prevented — go back to
the EC and challenge a different assumption. The direction may need
to change. This is not failure; it's rigor.
